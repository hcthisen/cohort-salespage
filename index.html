    Tier 1 – Pure Frontend Microapps

    No backend. Super simple. Surprisingly powerful.

    You’ll build small tools with just HTML, CSS, and JavaScript, powered by ChatGPT and microapp.live.

    Examples:

    • A teleprompter you can use for video recording.
    • A website value calculator.
    • A terrace builder or solar ROI calculator.

    Perfect if you want to ship something useful quickly and learn the basics of frontend and deployment without worrying about databases yet.

    Tier 2 – Frontend + Automation Backend

    When you need AI, APIs, and small automations.

    You’ll connect your frontend to an automation backend powered by n8n and whatever API services the app needs. This is ideal for webtools that process data, talk to APIs, or call AI.

    Examples:

    • A web tool that uses AI to summarize YouTube transcripts.
    • A selector tool like the aquaculture UV model selector for desmi.com.
    • Forms and tools that trigger workflows and send outputs to users.

    You’ll learn how to glue together a smart frontend with powerful automation backends, without overbuilding infrastructure.

    Tier 3 – Full Web App

    User accounts, databases, automations, and more.

    For those ready to go deeper, we’ll build full web apps with user logins, databases, and automation integrations. Here we bring in VPS hosting with backup, Supabase, GitHub, and GPT Codex.

    Example:

    • journal.vet, an app built for a veterinarian clinic with user accounts and real-world workflows.

    You’ll understand how to build focused, real web apps, not huge platforms, but specialized tools that are actually used in specific niches.
